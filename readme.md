# VectorJs

A 2D Vector libary for JavaScript. That's it!

## Installation

`npm i vectorjs`

## Usage

```javascript
var Vector = require('vectorjs');

var a = new Vector(2, 2);
var b = new Vector(5, 3);

var ab = a.add(b);

console.log(a, b);
// => { x: 7, y: 5 }
```
